Rules and Regulations

• To book more then one seat you need to add co-passenger detail.Otherwise you not able to do the booking.

• Adding Co-passenger detail is one time activity.You can use same information multiple time for seat booking.

• To add co-passenger detail go to Passenger Info link or click here.

• If more than one bus is scheduled for a particular time, the bookings for the second/third bus will be open only after first bus is 75% reserved.

• Seat Numbers are indicative and user can occupy any seat maintaining social distancing in view of COVID-19.
